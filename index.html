DESIGN AND ANALYSIS OF ALGORITHMS PORTFOLIO

ABOUT COURSE

Topic Description
1. Foundations Design Philosophy and Intuitions, Space and Time Complexities, Order of an Algorithm, Problem patterns: Recursion, Iteration, Backtracking
2. Principles Pruning, Edge Relaxation, Sets, Traversals, Prefix and Suffix, Union-Find, Hashing and Other Principles
3. Structured Data Graphs and Trees, Tries, AVL Trees, 2-3 Trees, Red-Black Trees, DFS, BFS, Heap, Array Query, Sparse Table, Segment Trees, Fenwick Trees, Skip Lists
4. Sorting This cluster focuses on the fundamental algorithms and techniques for sorting and searching, which are critical to efficient data processing and retrieval. Students will explore various sorting methods, such as comparison-based algorithms and specialized techniques, along with optimized searching strategies. Algorithms like quick sort, merge sort, bubble sort, insertion sort, selection sort, and heap sort are covered.
5. Graph Algorithms Included Dijkstra’s algorithm, Bellman-Ford, DFS, BFS, Prim’s, and Kruskal’s algorithms for network-related problems.

ABOUT PROJECT

Transportation: Enhancing the E-transportation sites and Functionalities

Problem Statement

This project aims to improve the digital interface and functionalities of e-transportation systems, making them more user-friendly and efficient. It includes developing features such as ticket scanning, seat allocation, traffic control, and waste management systems. The goal is to provide seamless, automated solutions for various transportation needs.

BRTS Ticket Scanning
Develop a robust system to verify tickets using QR codes, RFID cards, or physical passes, ensuring secure and seamless travel experiences.
Traffic Signal Control
Implement intelligent traffic signal systems that adapt to real-time traffic flow, reducing congestion and travel delays.
Seat Allotment for Trains/Buses
Create an efficient seat allocation algorithm that optimally assigns seats based on booking preferences and availability.
Shortest Distance Between Two Stops
Design a solution to compute the minimum travel distance or time between any two stops on a route using graph-based algorithms.
Transportation of Trash
Optimize trash transportation routes to minimize distance and cost, enabling efficient waste management.
Booked Ticket Search
Provide a fast and user-friendly search functionality to retrieve details of booked tickets accurately.
